				that information. A web page is a document that contains or references various kinds of data,
				such as text,images,graphics,programs, also containing links(to other web pages)using a 
				point-and-click interface, a website is a collection of related web pages. Web browser(requests
				web pages), then a web server(disigned to respond,responds to  that specific request) typically

				consists of seperate elements such as images.
